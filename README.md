# Rust Container Engine (Mini-Docker)

## Presentation du projet

Ce projet a pour vocation la conception et le développement d'une ébauche de moteur de conteneurisation en Rust. L'ambition n'est pas de produire un outil complexe, mais de tenter de reconstruire, à mon échelle, les mécanismes fondamentaux de l'isolation logicielle pour en comprendre le fonctionnement.

La stack technique sera définie de manière itérative au fur et à mesure de mes découvertes sur le langage Rust et les systèmes Unix.

## Objectifs techniques et apprentissages

Le développement de ce "Mini-Docker" est un terrain d'exercice pour explorer certains concepts de programmation système :

- **Démystifier le noyau Linux :** Apprendre comment fonctionnent l'isolation via les Namespaces, la gestion des ressources avec les Cgroups et le changement de racine via Chroot.
- **Apprivoiser Rust :** Pratiquer les concepts de sécurité mémoire, de gestion d'erreurs et d'appels système.
- **Comprendre les ressources :** Observer comment un logiciel interagit avec la mémoire et les processus.

## Methodologie et philosophie

Ce dépôt est conçu comme un carnet d'apprentissage personnel sur le long terme :

- **Un projet au long cours :** Ce travail se développera sur plusieurs années, au rythme de mes études. Il me servira de support pratique pour mieux assimiler les notions théoriques abordées en cours.
- **Sources d'appui :** Je m'appuie sur le guide officiel de Rust et sur l'utilisation de l'IA comme un mentor pour m'aider à tracer une ligne de progression cohérente, sans brûler les étapes, la génération de code est proscrite par ma propre personne.
- **Apprendre à coder proprement :** Mon but est de comprendre chaque ligne de code écrite et d'intégrer les principes du Clean Code dès mes premiers pas.

## Demarche personnelle

Ce projet accompagne ma transition vers l'informatique et ma préparation à une admission en BUT Informatique. Plus qu'un résultat fini, c'est le témoignage de ma curiosité pour les langages de bas niveau et de ma volonté d'apprendre avec patience et attention aux détails.
